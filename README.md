# Barrel_shifter_4
The basic shifter in digital circuits is designed using sequential circuits. Sequential circuits take multiple clock cycles to shift the required number of data. This Shifting can also be done using a single clock cycle, which is called as Barrel Shifter. This is basically a Combinational circuit. I've different modes (which says which shift to take) {mode '00'-logic left shift, mode '01'- logic right shift, mode '10'- rotate right, mode '11' - rotate left}, and provided shift amount too {'00' no shift, '01' shift by 1 bit, '10' shift by 2 bits, '11' shift by 3 bits}.
Example video {provided}: input data - '1011' = 'B'; mode 00 shift_amt 01 = 0110, shift_amt 11 = 1000; mode 10 shift_amt 01 = 0111, shift_amt 11 = 1101; mode 01 shift_amt 01 = 0101, shift_amt 11 = 0001; mode 11 shift_amt 01 = 1101, shift_amt 11 = 0111.
